For this challenge, Rakuten France provides around 99,000 product listings in CSV format, including the training set (84,916 samples) and the test set (13,812 samples).  
<br>

The data is organized into four separate datasets:

- **« X_train_update.csv »**: training samples with text fields (title and description) and the associated image file name  
- **« y_train_CVw08PX.csv »**: contains the target variable (**prdtypecode**)  
- **« X_test_update.csv »**: test samples used for submitting final predictions  
- **« images.zip »**: a zip file containing all product images:  
  - « image_train »: 84,916 images for the **training** set  
  - « image_test »: 13,812 images for the **test** set
