This project was initially developed as part of a challenge proposed by [Rakuten](https://challengedata.ens.fr/participants/challenges/35/) during our Data Scientist training at [DataScientest](https://datascientest.com/).

The challenge focused on the **classification of e-commerce products** using both **text data** (product title and description) and **image data**, with the goal of predicting the product type code.

A **baseline model** was provided by the challenge organizers:
- RNN on text data: **0.8113 weighted F1-score**
- ResNet on image data: **0.5534 weighted F1-score**

The **weighted F1-score** was the official evaluation metric for model performance.
